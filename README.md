# マネーマネー記帳

マネーマネー記帳：モバイル向けのシングルページアプリケーション

## 特徴

- 迅速な記帳
- 記帳、タグ付け、統計などの機能がある

## 🛠️ クイックスタート

```bash
# リポジトリをクローン
git clone https://github.com/NansenHo/money-react.git
cd money-react

# パッケージをインストール
yarn add

# 開発環境を起動
yarn start

# ビルド
yarn run build
```

## 💻 テックスタック

- TypeScript
- React
- ReactRouter
- Custom Hooks
- Styled Components
